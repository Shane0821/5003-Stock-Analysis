# 5003-Project

## Requirement
- docker desktop installed
- python 3.11.9
    - python kafka
    - pyspark
    - selenium
    - webdriver_manager
- jdk 17.0.10 or later

## Commands
To start kafka and spark cluster, execute:
```
./start_service.sh
```

To stop kafka and spark cluster, execute:
```
./stop_service.sh
```
